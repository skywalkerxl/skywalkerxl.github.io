区别与相关之处

这三者分别是什么？

从根本上来说，localhost是域名，127.0.0.1是ip地址，最然人分解的地方也许是本机 ip，因为这涉及到网卡的概念 ，本机网卡分为三块：有线网卡、无线网卡、虚拟网卡，具体的本机地址为多少取决于是与哪个网络接口绑定，这样可以供其它设备访问你的设备。

为什么这三者会联系在一起？

在hosts文件中，定义了localhost=127.0.0.1，这是一个容易混淆的地方。
